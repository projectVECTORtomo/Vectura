# Vectura - Professional LaTeX Table Workbench
### 🚀 Part of Project VECTOR | Developed by tomo

Vectura is a high-precision LaTeX generation engine designed for STEM researchers and students. It transforms raw data into publication-ready LaTeX tables with academic rigor.

「研究を、美しく。データを、一瞬で。」  
Vecturaは、STEM領域の学術規約を網羅した、エンジニアのためのプロフェッショナルなLaTeXテーブル・ワークベンチです。

---

## 🛠 Features / 主な機能

* **Mathematical Precision (数学的精度):**
    * Significant Figures Handling (有効数字の自動処理): 物理・工学実験の基準に適した数値丸めを行います。
    * Error Propagation (誤差表示): `1.23 +- 0.05` のような入力を LaTeX の $\pm$ 形式へ自動変換します。
* **Academic Standards (学術基準への準拠):**
    * **Vector Bold Formatter:** 変数名を自動的に $\mathbf{V}$ のようなベクトル太字形式で出力可能です。
    * **siunitx & booktabs Support:** `siunitx` による数値整形や、`booktabs` による美しい水平線の制御に対応しています。
* **Professional UI/UX:**
    * **Desktop Optimized:** PCでの長時間の作業に最適な、洗練されたデスクトップ・レイアウト。
    * **Live Synchronization:** 入力データと設定が即座に出力コードに反映されます。

---

## 📂 File Structure / ファイル構成

拡張性とメンテナンス性を重視した「関心の分離（Separation of Concerns）」に基づく構成を採用しています。



* `index.html`: アプリケーションの構造とUI定義。
* `style.css`: プレミアム・ダークテーマおよびコンポーネントのデザイン 。
* `script.js`: データ解析、有効数字計算、LaTeX生成ロジック。

---

## 🚀 Getting Started / 使い方

1.  本リポジトリをクローンまたはダウンロードします。
2.  すべてのファイル（`index.html`, `style.css`, `script.js`）を同じフォルダに配置します。
3.  `index.html` をブラウザで開くだけで使用可能です。

**Deployment:** GitHub Pages 等にアップロードすることで、自分専用のワークベンチをオンラインで公開できます。
